Controls:

Use the Arrow Keys to move your player.

Use the Space key to shoot. You will shoot a projectile in the direction you are facing.


Instructions

At the start of the game a random level is selected.

Shoot as many enemies as possible before dying.


Click on the game screen area to resolve any keyboard input issues.
