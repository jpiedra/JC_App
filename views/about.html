About JavaCrypt

JavaCrypt is a small game written in JavaScript. It utilizes a MongoDB server to store small text documents to represent possible levels to generate when the game is started. Users can submit their own levels. This application runs on the MEAN stack.


You can learn more about the author, Jonathan Piedra, at his homepage: jpiedra.github.io
